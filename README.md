# Student_Grade_Prediction

## Overview
   This Project illustrates the machine learning technique to predict the student grades

## Dataset Used
   Kaggle Student Performance Dataset

## Objective
   The objective of this project is to predict student grades based on various academic and demographic factors using Linear Regression.

## Machine Learning Model Used
   Linear Regression Model

## Evaluation Metrics
   1) R2 score
   2) Mean Squared Error

## Results
   By Using Linear Regression, we get r2 score and mean squared error. A comparison also done between actual grades and predicted grades. The result indicates the    models effectiveness in students academic performance.

## Technologies Used
   1) Python
   2) Numpy
   3) Pandas
   4) Matplotlib
   5) Scikit-learn
